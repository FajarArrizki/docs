## Current Implementation Notes

| Area | Current state |
| --- | --- |
| charging asset | SPL token selected by the user profile |
| per-user ledger | `SpendingProfile` |
| delegated execution | `DelegatedSigner` PDA |
| immutable receipt | `AiUsageRecord` |
| model ids longer than 32 bytes | seeded with deterministic on-chain alias |

## Useful scripts

| Script | Purpose |
| --- | --- |
| `scripts/model-registry-helper.js` | resolve source model ids to on-chain ids |
| `scripts/verify-model-registry.js` | verify seeded registry accounts exist on-chain |
| `scripts/generate-model-registry-doc.js` | regenerate docs catalog page |
