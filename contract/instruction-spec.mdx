## Admin Instructions (7)

### 1. initialize_config

**Purpose**: Initialize platform configuration (one-time setup)

**Signers**:
- `authority` (mut) - Platform admin who will control config

**Accounts**:
```rust
#[account(init, payer = authority, space = PlatformConfig::LEN, seeds = [CONFIG_SEED], bump)]
pub config: Account<'info, PlatformConfig>

#[account(mut, seeds = [FEE_VAULT_SEED], bump)]
pub fee_vault: AccountInfo<'info>  // CHECK: Fee vault PDA

#[account(mut)]
pub authority: Signer<'info>

pub system_program: Program<'info, System>
```

**Parameters**:
- `platform_fee_bps: u16` - Platform fee in basis points (500 = 5%)
- `default_markup_bps: u16` - Default backend markup (500 = 5%)
- `backend_authority: Pubkey` - Backend wallet authorized to record usage

**Constraints**:
- `platform_fee_bps <= MAX_PLATFORM_FEE_BPS` (2000 = 20%)
- `default_markup_bps <= MAX_MARKUP_BPS` (5000 = 50%)
- Config PDA must not exist (init constraint)

**State Changes**:
```rust
config.authority = authority.key()
config.backend_authority = backend_authority
config.platform_fee_bps = platform_fee_bps
config.default_markup_bps = default_markup_bps
config.fee_vault = fee_vault.key()
config.total_fees_collected = 0
config.total_markup_collected = 0
config.is_paused = false
config.bump = ctx.bumps.config
config.fee_vault_bump = ctx.bumps.fee_vault
```

**Events**: None

**Failure Cases**:
- `InvalidFeePercentage` - platform_fee_bps > 2000
- `InvalidMarkupPercentage` - default_markup_bps > 5000
- `AccountAlreadyInitialized` - Config already exists
- Insufficient SOL for rent

## Model Registry Instructions (3)

### 8. register_model

**Purpose**: Register new AI model in registry

**Signers**:
- `authority` (mut) - Platform admin (pays for account creation)

**Accounts**:
```rust
#[account(
    init,
    payer = authority,
    space = ModelRegistry::space(model_id.len(), provider.len(), features.len()),
    seeds = [MODEL_REGISTRY_SEED, model_id.as_bytes()],
    bump
)]
pub model_registry: Account<'info, ModelRegistry>

#[account(mut)]
pub authority: Signer<'info>

pub system_program: Program<'info, System>
```

**Parameters**:
- `model_id: String` - Model identifier (e.g., "gpt-4-turbo")
- `provider: String` - Provider name (e.g., "openai")
- `base_cost_per_token: u64` - Suggested cost per token (lamports)
- `features: String` - Comma-separated features (e.g., "chat,vision")

**Constraints**:
- `model_id.len() > 0 && model_id.len() <= MAX_MODEL_ID_LEN` (64)
- `provider.len() > 0 && provider.len() <= MAX_PROVIDER_LEN` (32)
- `features.len() <= MAX_FEATURES_LEN` (256)
- Model PDA must not exist (init constraint)

**State Changes**:
```rust
model_registry.model_id = model_id
model_registry.provider = provider
model_registry.custom_contract = None
model_registry.base_cost_per_token = base_cost_per_token
model_registry.is_active = true
model_registry.is_verified = true
model_registry.features = features
model_registry.total_usage_count = 0
model_registry.total_tokens_processed = 0
model_registry.created_at = clock.unix_timestamp
model_registry.updated_at = clock.unix_timestamp
model_registry.bump = ctx.bumps.model_registry
```

**Events**:
```rust
emit!(ModelRegistered {
    model_id,
    provider,
    base_cost_per_token,
    timestamp: clock.unix_timestamp,
})
```

**Failure Cases**:
- `InvalidModelId` - model_id empty or too long
- `InvalidModelId` - provider empty or too long (reuses error)
- `ModelAlreadyRegistered` - Model PDA already exists
- Insufficient SOL for rent

## Delegation Instructions (2)

### 11. create_delegated_signer

**Purpose**: User creates delegation for backend to auto-sign usage

**Signers**:
- `owner` (mut) - User creating delegation (pays for account)

**Accounts**:
```rust
#[account(
    init,
    payer = owner,
    space = DelegatedSigner::LEN,
    seeds = [DELEGATED_SIGNER_SEED, owner.key().as_ref(), delegate.key().as_ref()],
    bump
)]
pub delegated_signer: Account<'info, DelegatedSigner>

#[account(mut)]
pub owner: Signer<'info>

pub delegate: AccountInfo<'info>  // CHECK: Delegate pubkey (no signature required)

pub system_program: Program<'info, System>
```

**Parameters**:
- `expiry_duration: i64` - Duration in seconds until expiry
- `spending_limit: u64` - Max lamports delegate can spend

**Constraints**:
- `expiry_duration >= MIN_DELEGATION_DURATION` (60 seconds)
- `expiry_duration <= MAX_DELEGATION_DURATION` (30 days)
- `spending_limit <= MAX_SPENDING_LIMIT` (1000 SOL)
- Delegation PDA must not exist (init constraint)

**State Changes**:
```rust
delegated_signer.owner = owner.key()
delegated_signer.delegated_pubkey = delegate.key()
delegated_signer.created_at = clock.unix_timestamp
delegated_signer.expires_at = clock.unix_timestamp + expiry_duration
delegated_signer.spending_limit = spending_limit
delegated_signer.spent_amount = 0
delegated_signer.is_active = true
delegated_signer.bump = ctx.bumps.delegated_signer
```

**Events**: None

**Failure Cases**:
- `InvalidDelegationDuration` - Duration out of range
- `SpendingLimitExceeded` - spending_limit > MAX_SPENDING_LIMIT
- Delegation already exists for this owner-delegate pair
- Insufficient SOL for rent

## AI Usage Instructions (2)

### 16. record_ai_usage

**Purpose**: Record AI usage with user signature (no delegation)

**Signers**:
- `user` (mut) - Vault owner (pays for usage record account)

**Accounts**:
```rust
#[account(
    init,
    payer = user,
    space = AiUsageRecord::space(model_id.len(), usage_type.len()),
    seeds = [AI_USAGE_SEED, vault.key().as_ref(), &clock.unix_timestamp.to_le_bytes()],
    bump
)]
pub usage_record: Account<'info, AiUsageRecord>

#[account(
    mut,
    seeds = [VAULT_SEED, user.key().as_ref()],
    bump = vault.bump,
    has_one = owner
)]
pub vault: Account<'info, Vault>

#[account(mut, seeds = [CONFIG_SEED], bump = config.bump)]
pub config: Account<'info, PlatformConfig>

#[account(mut, seeds = [FEE_VAULT_SEED], bump = config.fee_vault_bump)]
pub fee_vault: AccountInfo<'info>  // CHECK: Fee vault PDA

#[account(
    mut,
    seeds = [MODEL_REGISTRY_SEED, model_id.as_bytes()],
    bump = model_registry.bump
)]
pub model_registry: Option<Account<'info, ModelRegistry>>

pub owner: AccountInfo<'info>  // CHECK: Vault owner (for validation)

#[account(mut)]
pub user: Signer<'info>

pub system_program: Program<'info, System>
```

**Parameters**:
- `model_id: String` - Model identifier
- `base_cost: u64` - Model/provider cost (lamports)
- `service_cost: u64` - Monitoring or backend service cost (lamports)
- `usage_type: String` - "text", "image", "audio", "video"
- `tokens_used: u64` - Token count
- `markup_bps: Option<u16>` - Override markup (or use default)

**Constraints**:
- `!config.is_paused`
- `base_cost > 0`
- `model_id.len() > 0`
- If model_registry provided: `model_registry.is_active == true`
- `markup_bps.unwrap_or(config.default_markup_bps) <= MAX_MARKUP_BPS`
- `vault.balance >= total_charged`
- `user` must match `vault.owner`

**State Changes**:
```rust
// Calculate fees
markup = markup_bps.unwrap_or(config.default_markup_bps)
chargeable_cost = base_cost + service_cost
markup_amount = chargeable_cost * markup / 10000
cost_after_markup = chargeable_cost + markup_amount
platform_fee = cost_after_markup * config.platform_fee_bps / 10000
total_charged = cost_after_markup + platform_fee

// Transfer: vault -> fee_vault (platform_fee lamports)
vault.balance -= total_charged
vault.total_ai_usage += total_charged
config.total_fees_collected += platform_fee
config.total_markup_collected += markup_amount

// If model_registry provided
model_registry.total_usage_count += 1
model_registry.total_tokens_processed += tokens_used

// Create usage record
usage_record.user = user.key()
usage_record.vault = vault.key()
usage_record.delegated_signer = None
usage_record.model_id = model_id
usage_record.usage_type = usage_type
usage_record.tokens_used = tokens_used
usage_record.base_cost = base_cost
usage_record.service_cost = service_cost
usage_record.markup_bps = markup
usage_record.markup_amount = markup_amount
usage_record.platform_fee_bps = config.platform_fee_bps
usage_record.platform_fee_amount = platform_fee
usage_record.total_charged = total_charged
usage_record.timestamp = clock.unix_timestamp
usage_record.bump = ctx.bumps.usage_record
```

**Events**:
```rust
emit!(AiUsageRecorded {
    user: usage_record.user,
    vault: usage_record.vault,
    delegated_signer: None,
    model_id,
    usage_type,
    tokens_used,
    base_cost,
    service_cost,
    markup_bps: markup,
    markup_amount,
    platform_fee_bps: config.platform_fee_bps,
    platform_fee_amount: platform_fee,
    total_charged,
    timestamp: clock.unix_timestamp,
})
```

**Failure Cases**:
- `Unauthorized` - Platform is paused
- `InvalidUsageAmount` - base_cost == 0
- `InvalidModelId` - model_id empty
- `ModelNotActive` - Model registry provided but model inactive
- `InvalidMarkupPercentage` - markup > MAX_MARKUP_BPS
- `InsufficientVaultBalance` - vault.balance < total_charged
- `Unauthorized` - user != vault.owner
- Duplicate usage record (same vault + timestamp)
- Insufficient SOL for usage record rent

pub delegated_signer: Account<'info, DelegatedSigner>

#[account(mut, seeds = [CONFIG_SEED], bump = config.bump)]

// Create usage record

usage_record.vault = vault.key()
usage_record.delegated_signer = Some(delegated_signer.key())
usage_record.model_id = model_id
usage_record.usage_type = usage_type
usage_record.tokens_used = tokens_used
