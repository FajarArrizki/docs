## Admin Instructions

### initialize_config
Initialize platform configuration.

### update_platform_fee
Update platform fee percentage.

### update_default_markup
Update default markup percentage.

### update_authority
Transfer admin authority.

### update_backend_authority
Update backend charging authority.

### toggle_pause
Pause or unpause charging.

### claim_fees
Claim lamports held by the `fee_recipient` PDA.

This is separate from the SPL token charging flow, which sends usage charges into a runtime-supplied `fee_recipient_token_account` owned by `config.authority`.

## Spending Profile Instructions

### initialize_spending_profile
Create the user's spending profile.

**Accounts:**
- `spending_profile` - SpendingProfile PDA (init)
- `payment_mint` - SPL token mint used for charging
- `owner` - User signer
- `system_program` - System program

### approve_spending_delegate
Approve the delegated signer PDA as token spending delegate.

**Accounts:**
- `spending_profile` - SpendingProfile PDA
- `delegated_signer` - DelegatedSigner PDA
- `user_token_account` - User SPL token account
- `owner` - User signer
- `token_program` - SPL token program

### revoke_spending_delegate
Revoke delegated token spending approval.

### close_spending_profile
Close the user's spending profile.

**Accounts:**
- `spending_profile` - SpendingProfile PDA
- `user_token_account` - User SPL token account, which must no longer have a delegate approval
- `owner` - User signer

## Delegated Signer Instructions

### create_delegated_signer
Create delegated signer for backend.

### revoke_delegated_signer
Revoke delegated signer.

### close_delegated_signer
Close a revoked or expired delegated signer.

### register_model
Create a model registry entry as `config.authority`.

### update_model
Update a model registry entry as `config.authority`.

### deactivate_model
Deactivate a model registry entry as `config.authority`.

## AI Usage Instructions

### record_ai_usage
Record AI usage with direct user signature.

Accepted `usage_type`: `text`, `image`, `audio`, `video`.

**Main accounts:**
- `usage_record`
- `spending_profile`
- `config`
- `user_token_account`
- `fee_recipient_token_account`
- `payment_mint`
- `model_registry` (optional)
- `user`
- `token_program`
- `system_program`

### record_ai_usage_with_delegation
Record AI usage with backend signature and delegated token spend.

Accepted `usage_type`: `text`, `image`, `audio`, `video`.

**Main accounts:**
- `usage_record`
- `spending_profile`
- `delegated_signer`
- `config`
- `user_token_account`
- `fee_recipient_token_account`
- `payment_mint`
- `model_registry` (optional)
- `delegate`
- `token_program`
- `system_program`

## Events

### AiUsageRecorded

| Field | Meaning |
| --- | --- |
| `user` | end user |
| `spending_profile` | charging profile used |
| `delegated_signer` | delegation used, if any |
| `model_id` | model used |
| `base_cost` | model/provider cost |
| `service_cost` | monitoring or backend service cost |
| `markup_amount` | backend markup |
| `platform_fee_amount` | protocol fee |
| `total_charged` | final token amount charged |
