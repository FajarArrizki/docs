# OHLC Database

This document describes the historical OHLC storage layer used by the backend.

## Purpose

The OHLC database is used to persist candlestick data for charting and historical analysis workflows.

## Typical Uses

- chart history
- exchange-backed OHLC retrieval
- cached historical market context
